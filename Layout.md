# Layout do Formulário

Campos que o formulário deve ter:

## Cabeçalho

- Número da Nota
- Data de Emissão
- Chave de Acesso

## Emitente

- Nome
- CNPJ

## Destinatário

- Nome
- CPF/CNPJ

## Itens

Tabela com:

- Descrição
- Quantidade
- Valor Unitário
- Valor Total

## Totais

- Valor dos Produtos
- Valor Total da Nota

## Rodapé

- Assinatura
- Data
